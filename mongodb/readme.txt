To work with mongo db in yaksha, follow the steps:
1. select terminal menu, select open terminal in specific container and select Mongo
2. One mongo terminal is opened, switch to your work space
    cd <work_space>
3. To connect to mongo server, 
   mongo --host 127.0.0.1 --port 27017 -u user -p password
   --authenticationDatabase guestbook
4. There is only one default database called guestbook.
